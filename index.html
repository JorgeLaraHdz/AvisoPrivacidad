Aviso de Privacidad

Última actualización: [26 de noviembre de 2024]

La presente política de privacidad describe cómo se recopila, utiliza y protege la información de los usuarios de nuestra aplicación, diseñada para ofrecer noticias y contenido relacionado con Pokémon. Esta app utiliza la API de NewsAPI, la PokeAPI, y los servicios de Firebase.

Responsable del Tratamiento de Datos

El responsable del tratamiento de los datos es el desarrollador de esta aplicación. Para cualquier consulta, puedes contactarnos en: mentorlinksr@outlook.com

Información Recopilada

Nuestra aplicación no recopila información personal directamente. Sin embargo, utiliza servicios externos que podrían recopilar datos de manera automática:

  1. NewsAPI: Para obtener noticias relacionadas con Pokémon.
  2. PokeAPI: Para acceder a información sobre personajes, estadísticas y datos del universo Pokémon.
  3. Firebase:
    • Para gestionar funciones internas como autenticación, almacenamiento de datos y monitoreo de rendimiento.
    • En caso de habilitar notificaciones push, Firebase recopila un identificador anónimo de dispositivo para entregar los mensajes.

Uso de la Información

La información recopilada a través de terceros se utiliza únicamente para:

  • Proveer contenido relevante, como noticias y datos del universo Pokémon.
  • Mejorar el rendimiento y la funcionalidad de la aplicación.
  • Enviar notificaciones push si el usuario otorga permiso para recibirlas.

Almacenamiento y Protección de Datos

Los datos procesados por terceros (Firebase, NewsAPI, PokeAPI) están sujetos a las políticas de privacidad de cada servicio:

  • Política de privacidad de Firebase.
  • Política de privacidad de NewsAPI.
  • Política de privacidad de PokeAPI.

Permisos

Nuestra aplicación solicita únicamente el permiso para enviar notificaciones push. Este permiso es opcional y puede ser gestionado desde la configuración del dispositivo en cualquier momento.

Derechos del Usuario

Como usuario, tienes derecho a:

  • Revocar el permiso de notificaciones push desde la configuración del dispositivo.
  • Consultar y solicitar la eliminación de cualquier dato personal almacenado por servicios externos (si aplica).

Cambios en el Aviso de Privacidad

Nos reservamos el derecho de actualizar este aviso de privacidad. Te notificaremos sobre cualquier cambio significativo a través de la aplicación.
